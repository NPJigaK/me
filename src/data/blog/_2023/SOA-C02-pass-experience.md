この度、AWS Certified Machine Learning - Specialty（MLS-C01）試験に無事合格できましたので、その体験を共有します。

![AWS](@/assets/images/blog/2021/AWS.JPG)

## 目次

## AWS Certified Machine Learning - Specialtyとは？

この認定試験はAWS上で機械学習（ML）ソリューションを設計、実装、管理する能力を評価する試験です。

主に以下の内容が含まれています。

- 機械学習モデルの設計と実装
- MLソリューションの運用および管理
- データ準備と特徴量エンジニアリング

## 試験結果

- 得点：845/1000（合格点750点）

自己採点よりも良いスコアで安心しました。

## 試験準備

- 勉強期間：約6週間
- 使用教材：UdemyのAWS Machine Learning Specialty対策コース、AWS公式のドキュメント（特にAmazon SageMaker）

Udemyのコースで基本知識を固め、その後AWS公式ドキュメントでAmazon SageMakerやRekognition、Comprehendなどのサービスを深く掘り下げて学びました。

模擬試験を3回繰り返し、間違えた箇所を特に重点的に復習しました。

## 試験の感想

試験では具体的なシナリオベースの問題が多く、特にAmazon SageMakerを中心としたMLソリューションの詳細な理解が求められました。モデルのトレーニングと推論の最適化に関する問題が多かった印象です。

データ準備や特徴量エンジニアリングの知識も問われ、実務経験がなくても公式ドキュメントの事例を深く理解しておくと解答しやすいと感じました。

## 最後に

AWSのMachine Learning試験は実務に即した深い知識が求められるため難易度は高めですが、機械学習の専門知識が非常に深まります。

機械学習エンジニアを目指す方や、AWSの機械学習サービスを本格的に利用したい方には特におすすめです。

会社が試験費用を支援してくれる場合は、ぜひチャレンジしてみてください。

これから受験される方々の参考になれば幸いです。
