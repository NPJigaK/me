AWS Certified Data Analytics - Specialty（DAS-C01）試験に合格しましたので、その体験をシェアします。

![AWS](@/assets/images/blog/2021/AWS.JPG)

## 目次

## AWS Certified Data Analytics - Specialtyとは？

この試験は、AWSを活用したデータ分析ソリューションの設計、構築、保守に関する専門知識を評価します。具体的には以下の分野が含まれています。

- データ収集とデータ管理
- データ処理と分析
- データの可視化

## 試験結果

- 得点：870/1000（合格点750点）

自己評価よりも高いスコアが出て安心しました。

## 試験準備

- 勉強期間：約5週間
- 使用教材：AWS公式ドキュメント、UdemyのData Analytics Specialty対策コース

特にAWS Glue、Amazon Athena、Amazon Redshift、Amazon QuickSightの公式ドキュメントを徹底的に読み込みました。また、Udemyの対策コースでは模擬試験を繰り返し、特に間違った問題を深掘りして復習しました。

## 試験の感想

試験問題の多くがシナリオベースで、具体的なデータ分析の要件や最適なソリューションを選択する能力が求められました。特にAWS GlueとAmazon Redshiftに関する深い知識が役立ちました。

また、データの可視化や分析結果の解釈に関する問題も出題され、QuickSightや分析プロセスの理解が試されました。

## 最後に

AWS Data Analytics試験は実践的なスキルを高めることができるため、データ分析やビッグデータに興味がある方には非常におすすめです。

会社で資格取得のサポート制度がある場合、ぜひこの機会に挑戦してみてはいかがでしょうか。
